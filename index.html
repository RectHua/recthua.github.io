什么是 NetExplore

一款Win11上的IE兼容插件

NetExplore是我们用VB6开发的IE兼容插件，允许用户在 Windows 11 操作系统上以 IE7/8/9/10/11 的视图来查看网页。NetExplore 完全免费，不必担心任何费用问题。

源代码全部开源

所有NE源代码均可下载

NetExplore NeXT 2（即最新版本）为我们基于 Microsoft Visual Basic 6.0 构建，其全部源代码已于 GitHub 上公开，遵循 GPL-3.0 开源协议，您可以直接下载、自行编译，不必担心安全问题。


 源代码（GitHub）

支持IE的基本功能

支持F12开发人员工具

NetExplore拥有IE的历史记录、收藏夹、无痕模式、文件下载等基本功能,能够满足基本上网需求。

NetExplore允许您在不同版本的IE内核（IE-7/8/9/10/11）之间自由切换以适应不同的网页。

NetExplore还能使用IE的F12开发人员工具。
